# DescentImageGenerator

Simple yet efficient image rendering for landing trajectories.

# Key features

- [ ] Simple and robust config format
- [ ] Hackable renderer for comple use cases
- [ ] Meaningful examples

# Supported features

- [ ] Spherical DEM
- [ ] Fused DEM support
- [ ] Trajectory from TESOA reader
- [ ] Rock meshes loading and projection 
- [ ] Metadata export (depth map, normal map, label map, ...)
- [ ] LIDAR rendering

# Usage

- CLI example in descentimagegenerator/main.cpp:
```
python -m descentimagegenerator.main -c <json_config> [-d]
```

- Library examples in examples/
